# UI Shell Changelog

## 0.1.0 - 2026-04-21

- Established `@easydesign/ui-shell` as the reusable host layer between the EasyDesign component library and product apps.
- Added shell adapters for auth/session, permissions, feature flags, tenant context, notifications, deep links, analytics, persistence, assets, and adaptive runtime.
- Added inventory-app and ops-console consumer proof coverage plus shell guardrails for public entrypoints.

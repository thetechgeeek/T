# Ops Console Example

This workspace is the second consumer proof for the reusable UI packages.

- Imports `@easydesign/design-system` and `@easydesign/ui-shell` through public package entrypoints only.
- Demonstrates a different tenant, permissions, feature flags, deep-link policy, and notification feed from the inventory app.
- Serves as the go/no-go example before moving the UI packages into a separate repo.
